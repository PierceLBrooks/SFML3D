SFML3D - Simple and Fast Multimedia Library
=========================================

SFML3D is a simple, fast, cross-platform and object-oriented multimedia API. It provides access to windowing, graphics,
audio and network.
It is written in C++, and has bindings for various languages such as C, .Net, Ruby, Python.

Download
--------

You can get the latest official release on SFML3D website (http://www.sfml3d-dev.org/download.php).
You can also get the current development version from the git repository (https://github.com/LaurentGomila/SFML3D).

Install
-------

Follow the instructions of the tutorials (http://www.sfml3d-dev.org/tutorials.php), there is one for each platform/compiler that SFML3D supports.

Learn
-----

There are several places to learn SFML3D:
* The official tutorials (http://www.sfml3d-dev.org/resources.php#tutorials)
* The online API documentation (http://www.sfml3d-dev.org/resources.php#documentation)
* The community wiki (https://github.com/LaurentGomila/SFML3D/wiki/)
* The community forum (http://en.sfml3d-dev.org/forums/) (or http://fr.sfml3d-dev.org/forums/ for french people)

Contribute
----------

SFML3D is an open-source project, and it needs your help to go on growing and improving.
Don't hesitate to post suggestions or bug reports on the forum (http://en.sfml3d-dev.org/forums/),
submit patches by e-mail, or post new bugs/features requests on the task tracker (https://github.com/LaurentGomila/SFML3D/issues/).
You can even fork the project on github, maintain your own version and send us pull requests periodically to merge your work.
